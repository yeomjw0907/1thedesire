# 욕망백서 문서 관리

이 폴더는 욕망백서 서비스의 기획, 법률, 개발 문서를 관리한다.

## 폴더 구조

- `planning/`: 서비스 기획, 포지셔닝, UX, 와이어프레임, 디자인 문서
- `legal/`: 이용약관, 운영정책, 포인트 및 환불 정책
- `development/`: 데이터베이스, SQL, 마이그레이션 설계 문서
