# 욕망백서 로고 에셋 규격

준비된 에셋(심볼·타이포·혼합형, 흰/검)을 **어떤 형식으로 전달하면 어디에 쓸지** 정리한 문서입니다.  
전달받은 파일은 `public/` 폴더에 넣어 사용합니다.

---

## 1. 한눈에 보기: 전달 형식 → 사용처

| 전달할 파일명 | 쓸 에셋 | 규격 | 사용처 (코드 위치) |
|---------------|---------|------|--------------------|
| **logo-loading.png** | 혼합형 또는 타이포 **가로**, **흰색** | 가로 240px 내외 (비율 유지) | 전역 로딩 화면 — `src/app/loading.tsx` (검정 배경) |
| **logo-login.png** | 혼합형 또는 타이포 **가로**, **흰색** | 가로 200px 내외 | 로그인/가입 상단 — `src/app/login/page.tsx` (선택, 현재는 텍스트 "욕망백서") |
| **icon-192.png** | **심볼(1:1)**, **흰색** | **192 × 192 px** | PWA manifest, Android 홈화면 아이콘 — `public/manifest.json` |
| **icon-512.png** | **심볼(1:1)**, **흰색** | **512 × 512 px** | PWA manifest, 스플래시 등 — `public/manifest.json` |
| **favicon.ico** | **심볼(1:1)**, **흰색** (또는 검정) | **32 × 32 px** (.ico) | 브라우저 탭 아이콘 (현재는 `src/app/icon.tsx` 텍스트 아이콘 사용 중) |

**검정 버전**은 흰/밝은 배경 화면이 생길 때 사용합니다. 예: `logo-loading-dark.png`, `logo-login-dark.png` — 필요 시 추가.

---

## 2. 보유 에셋별 → 전달 파일 매핑

| 보유 에셋 | 전달 시 파일명 | 사용처 요약 |
|-----------|----------------|-------------|
| **심볼 (1:1), 흰색** | `icon-192.png`, `icon-512.png`, `favicon.ico` | PWA 아이콘, 파비콘 |
| **타이포 로고 (가로), 흰색** | `logo-loading.png`, `logo-login.png` | 로딩, 로그인 |
| **혼합형 로고 (가로), 흰색** | `logo-loading.png`, `logo-login.png` | 로딩, 로그인 |
| **혼합형 로고 (세로), 흰색** | `logo-loading.png` (높이 48px 기준으로 배치 가능) | 로딩 (세로 레이아웃일 때) |
| **흰색 → 검정 버전** | `logo-*-dark.png` 등 | 흰/밝은 배경용 (필요 시) |

---

## 3. 포맷 및 공통 규격

| 항목 | 권장 |
|------|------|
| **포맷** | **PNG** (투명 배경 권장) 또는 **SVG**. 파비콘만 **.ico** |
| **색** | 검정 배경(#0B0B0D)용 = **흰색(밝은색)** 로고, 흰 배경용 = **검정** 로고 |
| **해상도** | PNG는 **2x(레티나)** 로 2배 크기 제작 후 1x로 사용 가능 (예: 192px 요청 시 384px 제작) |

---

## 4. public 폴더 구조 예시

```
public/
  logo-loading.png    # 로딩 — 가로형, 흰색 (혼합형 또는 타이포)
  logo-login.png      # (선택) 로그인/가입 — 가로형, 흰색
  icon-192.png        # PWA — 심볼 192×192, 흰색
  icon-512.png        # PWA — 심볼 512×512, 흰색
  favicon.ico         # 파비콘 — 심볼 32×32
```

---

## 5. 요약 체크리스트 (전달 시)

- [ ] **logo-loading.png** — 혼합형 또는 타이포 **가로**, **흰색**, 가로 240px 내외
- [ ] **icon-192.png** — **심볼(1:1)**, **흰색**, 192×192
- [ ] **icon-512.png** — **심볼(1:1)**, **흰색**, 512×512
- [ ] **favicon.ico** — **심볼(1:1)**, 32×32
- [ ] (선택) **logo-login.png** — 가로형, 흰색, 로그인/가입용

에셋 넣은 뒤 알려주시면 로딩 / manifest / 파비콘 연결까지 코드 반영해 드리면 됩니다.
