# 욕망백서 온보딩 / 가입 플로우 명세 v0.1

## 1. 목적

- 첫 진입부터 가입 완료까지의 흐름을 브랜드 기준으로 고정한다.
- 사용자가 부담 없이 진입하되, 서비스의 규칙과 분위기를 이해하게 만든다.

## 2. 온보딩의 역할

온보딩은 정보를 입력받는 절차가 아니다.  
욕망백서가 어떤 공간인지 이해시키고, 사용자가 이 공간의 결에 맞는 사람인지 확인하는 과정이다.

핵심 목표:

- 브랜드 분위기 전달
- 신뢰감 형성
- 과도한 노출이나 자극 없이 가입 전환 유도
- 가입 완료 후 즉시 피드 탐색으로 연결

## 3. 전체 플로우

```text
랜딩 진입
-> 시작하기
-> 간편 로그인
-> 가입 정보 입력
-> 약관/19세 체크
-> 가입 완료
-> 여성일 경우 270P 지급
-> 자동 소개글 생성 안내
-> 홈 피드 진입
```

## 4. 단계별 명세

### 4-1. 진입 화면

목적:

- 서비스 톤 전달
- 불필요한 긴장감 제거

구성:

- 로고
- 메인 카피
- 서브 카피
- 시작하기 버튼
- 보조 CTA `분위기 둘러보기`

추천 메인 카피:

- 분위기를 먼저 보고, 대화는 그 다음에

추천 서브 카피:

- 취향을 안전하게 나누는 익명 플랫폼입니다. 피드와 프로필을 먼저 보고, 요청이 수락된 뒤에만 대화를 시작하세요.

### 4-2. 로그인 화면

목적:

- 진입 장벽 최소화

구성:

- 간편 로그인 버튼
- 간단한 안내 문구

문구:

- 먼저 간단히 로그인해 주세요
- 가입 후에는 바로 분위기를 둘러볼 수 있습니다

### 4-3. 가입 정보 입력 화면

목적:

- 최소 필수 정보로 프로필을 완성
- 사용자가 `내 분위기`를 한 번 생각하게 만들기

필드:

- 닉네임
- 성별
- 연령대
- 지역
- 성향
- 자기소개

UI 구조:

```text
[상단]
진행 단계 표시

[본문]
헤드라인
필드 리스트

[하단]
다음 또는 가입 완료 버튼
```

헤드라인 후보:

- 이 공간에서의 분위기를 알려주세요
- 대화를 시작하기 전에 보여줄 정보를 입력해보세요

필드 힌트:

- 닉네임: 이 공간에서 사용할 이름
- 지역: 너무 자세하지 않게 설정해도 괜찮습니다
- 성향: 나를 설명하는 취향 한두 가지
- 자기소개: 대화를 시작하기 전에 보여주고 싶은 분위기

### 4-4. 약관 / 동의 화면

목적:

- 필수 동의를 명확히 받는다
- 경고형 문구가 아니라 규칙 안내형 문구로 전달한다

체크 항목:

- 본인은 19세 이상입니다
- 운영정책 및 이용약관에 동의합니다

보조 문구:

- 원치 않는 요청은 언제든 거절하거나 차단할 수 있습니다
- 외부 연락처 요구, 금전 요구, 협박은 신고 대상입니다

### 4-5. 가입 완료 화면

목적:

- 가입 완료 감각 제공
- 첫 액션으로 피드 탐색을 유도

구성:

- 완료 메시지
- 자동 소개글 안내
- 여성일 경우 270P 지급 안내
- 홈으로 이동 버튼

완료 문구:

- 이제 분위기를 둘러볼 수 있습니다
- 첫 소개글이 등록되었습니다. 언제든 수정하거나 삭제할 수 있습니다

여성 지급 문구:

- 가입 보너스로 270P가 지급되었습니다

## 5. 카피 톤 원칙

- 유치한 설렘 유도 금지
- 과도한 법률 경고 금지
- 서비스 규칙은 짧고 명확하게
- 가입은 `심문`이 아니라 `입장 정리`처럼 느껴져야 함

## 6. UI 스타일 원칙

- 진행 단계는 상단의 얇은 프로그레스 바로 표현
- 필드 간 간격은 넉넉히
- 검은 바탕 위에 진홍 버튼만 남발하지 않기
- 입력창은 매트한 surface 사용
- 버튼은 항상 한 단계만 강조

## 7. 예외 처리 UX

### 닉네임 중복

- 이미 사용 중인 이름입니다
- 다른 이름으로 조정해 주세요

### 필수값 누락

- 아직 입력하지 않은 정보가 있습니다

### 자기소개 미입력

- 한두 문장만으로도 괜찮습니다
- 대화를 시작하기 전에 보여주고 싶은 분위기를 적어주세요

### 동의 누락

- 가입을 완료하려면 필수 동의가 필요합니다

## 8. 첫 세션 유도

가입 직후 바로 해야 할 행동:

1. 홈 피드 보기
2. 내 프로필 확인
3. 자동 소개글 확인

가입 직후 노출할 수 있는 짧은 안내:

- 먼저 분위기를 둘러보고, 마음에 드는 사람의 프로필을 확인해보세요

## 9. 최종 판단

온보딩은 욕망백서의 첫 번째 안전장치다.

과도하게 무겁지도, 너무 가볍지도 않게  
`이 서비스는 조심스럽고 정리돼 있다`는 느낌을 주는 것이 중요하다.
