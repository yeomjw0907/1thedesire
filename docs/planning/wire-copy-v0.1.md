# 욕망백서 홈 / 프로필 / DM 텍스트 와이어카피 v0.1

## 1. 목적

- 주요 화면에서 실제로 노출될 텍스트를 와이어프레임 수준으로 정리한다.
- 디자인과 개발이 같은 문구 기준으로 작업할 수 있게 한다.

## 2. 홈 화면 와이어카피

### 상단

- 로고 또는 워드마크: `욕망백서`
- 히어로 문구: `분위기를 먼저 보고, 대화는 그 다음에`

### 탭

- `전체`
- `신규`

### 빈 상태

- `아직 올라온 글이 없습니다`
- `첫 분위기를 남겨보세요`

### 글쓰기 버튼

- `글쓰기`

### 게시글 카드 예시

```text
닉네임
30대 초반 · 서울
Sub · FWB
12분 전

처음엔 조용한 대화를 선호해요.
서로 분위기 맞는지 먼저 보는 편입니다.

[프로필 보기] [신고]
```

### 홈 안내 보조 문구

- `글과 분위기로 먼저 사람을 읽어보세요`

## 3. 프로필 화면 와이어카피

### 상단 액션

- `신고`
- `차단`

### 프로필 헤더 예시

```text
Chloe
20대 후반 · 경기 · 여성
Sub · FWB
```

### 상태 라인

- `최근 활동: 오늘`
- `받은 요청 수락 무료`

### 자기소개 섹션

- 타이틀: `소개`
- 본문 예시:
  `서로 말투와 분위기가 맞는지 먼저 보는 편이에요.
  급한 대화보다는 결이 맞는 사람이 좋습니다.`

### 최근 글 섹션

- 타이틀: `최근 분위기`

### DM CTA 영역

- 버튼: `대화 요청 보내기`
- 보조 문구 1: `요청 시 90P가 차감됩니다`
- 보조 문구 2: `거절 시 45P 환불 · 24시간 미응답 시 전액 환불`

## 4. DM 요청 팝업 와이어카피

### 제목

- `대화 요청 보내기`

### 대상 정보 예시

```text
Chloe
20대 후반 · 경기
Sub · FWB
```

### 정책 패널

- `요청 시 90P가 차감됩니다`
- `상대가 거절하면 45P가 환불됩니다`
- `24시간 내 응답이 없으면 전액 환불됩니다`
- `수락 후 대화는 무료입니다`

### 버튼

- `취소`
- `요청 보내기`

## 5. DM 목록 와이어카피

### 상단

- 타이틀: `DM`

### 탭

- `대기`
- `진행중`
- `종료`

```text
어제
```

## 6. 채팅방 와이어카피

### 상단

- `프로필 보기`
- `신고`
- `차단`

### 상태 배너 예시

#### pending

- `상대의 응답을 기다리는 중입니다`

#### agreed

- `대화를 시작할 수 있습니다`

#### declined

- `요청이 거절되어 45P가 환불되었습니다`

#### expired

- `요청이 만료되어 90P가 전액 환불되었습니다`

#### blocked

- `더 이상 이 대화를 이어갈 수 없습니다`

### 입력창 placeholder

- agreed 상태: `메시지를 입력하세요`
- 비활성 상태: `수락 후 대화를 시작할 수 있습니다`

## 7. 포인트 요약 와이어카피

### 상단

- `현재 보유 포인트`

### 안내 문구

- `새로운 대화 요청에만 포인트가 사용됩니다`
- `수락된 대화는 추가 과금 없이 이어집니다`

## 8. 신고 / 차단 와이어카피

### 신고 바텀시트

- 제목: `이 사용자를 신고할까요`
- 설명: `불편하거나 부적절한 행동이 있었다면 알려주세요`
- 버튼: `취소` / `신고하기`

### 차단 바텀시트

- 제목: `이 사용자를 차단할까요`
- 설명: `차단하면 서로의 프로필과 대화가 제한됩니다`
- 버튼: `취소` / `차단하기`

## 9. 최종 판단

욕망백서의 와이어카피는 자극적인 문장보다 `현재 상태를 명확히 설명하는 문장`이 더 중요하다.

사용자는 여기서 흥분만 원하지 않는다.  
내가 어디에 있고, 무엇이 열렸고, 무엇이 닫혔는지 분명히 알고 싶어 한다.
