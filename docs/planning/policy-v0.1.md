# 욕망백서 정책 문서 v0.1

## 1. 가입 정책

- 회원가입 시 필수 입력값은 `닉네임`, `성별`, `연령대`, `지역`, `성향`, `자기소개`다.
- 회원가입 시 `본인은 19세 이상입니다` 체크는 필수다.
- 회원가입 시 약관 및 운영정책 동의는 필수다.
- 여성 회원은 가입 시 270P를 기본 지급한다.
- 가입 완료 시 시스템이 기본 소개글을 자동 생성해 피드에 등록한다.
- 자동 생성 게시글 예시: `안녕하세요. [지역] 거주 [성향]입니다.`

## 2. 피드 정책

- 유저는 텍스트 게시글을 작성할 수 있다.
- 게시글에는 이미지 1장까지 첨부할 수 있다.
- 자동 생성된 첫 게시글도 일반 게시글처럼 노출된다.
- 유저는 자신의 게시글을 수정하거나 삭제할 수 있다.

## 3. DM 및 포인트 정책

- 다른 유저에게 DM 요청 시 90P가 즉시 차감된다.
- 상대방이 수락하기 전까지 채팅은 시작되지 않는다.
- 상대방이 수락하면 채팅방이 활성화된다.
- 상대방이 거절하면 45P가 환불된다.
- 상대방이 응답하지 않으면 24시간 후 요청 만료 처리하고 90P 전액 환불한다.
- 상대방이 차단한 경우 환불은 없다.
- 새로운 DM 요청을 보내는 행위에는 공통적으로 포인트가 필요하다.
- 여성 회원은 받은 DM을 수락하고 채팅을 시작할 때 추가 포인트가 들지 않는다.
- 남성 회원도 받은 DM을 수락하고 이미 열린 채팅방에서 답장할 때는 추가 포인트가 들지 않는다.

## 4. 성별별 포인트 정책

### 여성 회원

- 가입 시 270P를 기본 지급한다.
- 다른 유저에게 새로운 DM 요청을 보낼 때는 90P가 차감된다.
- 받은 DM을 수락하고 대화를 시작할 때는 무료다.
- 수락 이후 채팅방에서 메시지를 주고받는 행위는 무료다.

### 남성 회원

- 가입 시 기본 포인트 지급은 없다.
- 다른 유저에게 새로운 DM 요청을 보낼 때는 90P가 차감된다.
- 받은 DM을 수락하고 대화를 시작할 때는 무료다.
- 수락 이후 채팅방에서 메시지를 주고받는 행위는 무료다.

## 5. 채팅 정책

- 상호 합의가 완료된 뒤에만 메시지 입력이 가능하다.
- 상호 합의 전에는 채팅방이 생성되어도 입력창은 비활성 상태다.
- 차단된 유저 간에는 추가 메시지 전송이 불가하다.

## 6. 신고 및 제재 정책

- 유저는 프로필, 게시글, 채팅을 신고할 수 있다.
- 유저는 특정 유저를 차단할 수 있다.
- 차단 시 상호 노출과 DM 요청은 제한된다.
- 운영자는 신고 내용을 기준으로 `경고`, `일시 제한`, `영구 정지`를 할 수 있다.

## 7. 금지행위

- 미성년자 관련 내용
- 사칭
- 협박, 금전 갈취, 합의금 유도
- 개인정보 요구 및 유포
- 불법 촬영물, 유출물, 보복성 콘텐츠
- 스팸성 DM 반복 발송
- 강압적 오프라인 유도

## 8. 데이터 보관 정책 초안

- 프로필, 게시글, 포인트 거래 이력, 신고 이력은 저장한다.
- 합의 이벤트와 포인트 차감 이력은 별도 로그로 저장한다.
- 메시지 보관 기간은 90일로 한다.
- 탈퇴 시 삭제 범위와 로그 보존 범위는 미정이다.

## 9. 운영 우선순위

1. 미성년 의심 계정
2. 협박, 사기, 금전 유도
3. 강압적 메시지
4. 불법 촬영물 또는 유포 시도
5. 일반 분쟁
