# 욕망백서 신규 사용자 첫 7일 경험 설계 v0.1

## 1. 목적

- 가입 직후부터 첫 7일 동안 사용자가 어떻게 정착하는지 설계한다.
- 단순 재방문이 아니라 `서비스 분위기를 이해하고, 프로필을 채우고, 첫 요청 또는 첫 응답을 경험하는 것`을 목표로 한다.

## 2. 첫 7일의 핵심 목표

### Day 0

- 가입 완료
- 자동 소개글 확인
- 홈 피드 첫 탐색

### Day 1~2

- 내 프로필 보완
- 타인 프로필 탐색
- 첫 DM 요청 또는 첫 DM 응답 경험

### Day 3~4

- 피드 재방문
- 최근 글 작성 또는 프로필 업데이트
- DM 상태 확인

### Day 5~7

- 반복 방문 습관 형성
- 포인트 구조 이해
- 커뮤니티 분위기 적응

## 3. 핵심 원칙

- 첫 7일은 `기능 설명`보다 `분위기 적응`이 중요하다
- 유저를 급하게 결제로 밀지 않는다
- 여성과 남성의 첫 경험은 다르게 보완한다
- 첫 요청보다 먼저 `탐색의 편안함`을 주는 것이 중요하다

## 4. 단계별 설계

### Day 0: 첫 진입

사용자 상태:

- 막 가입함
- 아직 서비스의 분위기와 규칙을 체감하지 못함

보여줘야 할 것:

- 자동 소개글 생성 안내
- 홈 피드 진입
- 내 프로필 확인 유도

추천 문구:

- 이제 분위기를 둘러볼 수 있습니다
- 첫 소개글이 등록되었습니다. 언제든 수정하거나 삭제할 수 있습니다

유도 행동:

1. 홈 피드 보기
2. 내 프로필 확인
3. 최근 글 몇 개 읽기

### Day 1: 분위기 익숙해지기

사용자 상태:

- 아직 조심스럽고 관찰 위주

보여줘야 할 것:

- 홈 피드의 신규 글
- 비슷한 분위기의 프로필
- 자기소개 보완 유도

추천 문구:

- 먼저 글과 분위기로 사람을 읽어보세요
- 내 프로필을 조금 더 채워보세요

유도 행동:

- 프로필 보완
- 최근 글 읽기
- 성향 태그 확인

### Day 2: 첫 상호작용

사용자 상태:

- 누군가에게 요청을 보낼지, 받은 요청에 응답할지 고민하는 시기

여성 유저에게:

- 받은 요청 수락 무료 안내
- 270P 지급 상태 확인

남성 유저에게:

- 새 요청에만 90P 차감
- 거절/만료 환불 규칙 재안내

추천 문구:

- 분위기가 맞는 사람에게만 신중하게 요청해보세요
- 수락 후 대화는 무료입니다

### Day 3~4: 첫 경험 정착

사용자 상태:

- 첫 요청 또는 첫 응답 결과를 경험했을 가능성 높음

보여줘야 할 것:

- DM 상태 변화
- 환불 발생 시 내역 반영
- 피드 재방문 유도

추천 문구:

- 최근 올라온 분위기를 둘러보세요
- 요청 상태가 정리되었습니다

### Day 5~7: 습관 형성

사용자 상태:

- 다시 올 이유가 생기느냐가 중요

보여줘야 할 것:

- 피드의 지속적 흐름
- 내 프로필 보완 여지
- 포인트 구조의 이해

추천 문구:

- 이 공간의 흐름을 조금 더 천천히 읽어보세요
- 내 소개와 최근 글이 대화의 시작이 됩니다

## 5. 여성 / 남성 경험 차이

### 여성 유저 첫 7일

핵심:

- 안전감
- 통제감
- 받은 요청 수락 무료의 명확한 인지

중요 노출:

- 270P 지급
- 차단/신고 접근
- 받은 요청 수락 무료

### 남성 유저 첫 7일

핵심:

- 과금 구조의 납득
- 탐색 후 요청이라는 흐름 이해

중요 노출:

- 새 요청에만 90P 사용
- 거절/만료 환불 구조
- 수락 후 무료 대화

## 6. 첫 7일 알림 / 리마인드 포인트

### Day 0~1

- 첫 소개글 등록 안내
- 홈 피드 둘러보기 유도

### Day 2~3

- 프로필 보완 유도
- 받은 요청 또는 보낸 요청 상태 변화 알림

### Day 4~7

- 최근 글 흐름 알림
- 포인트/환불 정리 알림

## 7. 첫 7일에 피해야 할 것

- 가입 직후 과한 충전 유도
- 너무 잦은 푸시
- 기능을 한 번에 다 설명하는 온보딩
- 여성 메리트를 과도하게 광고처럼 반복 노출

## 8. 성공 기준

- Day 1 재방문
- 첫 3일 내 프로필 보완율
- 첫 3일 내 프로필 열람 수
- 첫 7일 내 첫 DM 요청 또는 첫 DM 수락 경험
- 자동 소개글 삭제율

## 9. 최종 판단

욕망백서의 첫 7일은 `바로 결제하게 만드는 구간`이 아니라  
`이 공간의 결을 이해하고, 천천히 관계를 시작하게 만드는 구간`이어야 한다.

그래야 이후 과금과 재방문도 자연스럽게 이어진다.
