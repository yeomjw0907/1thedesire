# 욕망백서 랜딩 페이지 명세 v0.1

## 1. 목적

- 랜딩 페이지를 브랜드 관문으로 설계한다.
- 광고성 문구 대신 서비스의 결, 신뢰, 방식이 드러나게 한다.

## 2. 랜딩의 역할

- 이 서비스가 어떤 공간인지 한 번에 이해시킨다
- 자극적 호기심보다 `신뢰 가능한 궁금증`을 만든다
- 가입 또는 둘러보기로 자연스럽게 이어지게 한다

## 3. 페이지 구조

```text
Hero
-> 핵심 가치 섹션
-> 어떻게 시작되는지 섹션
-> 포인트/환불 설명 섹션
-> 안전/운영 원칙 섹션
-> CTA 섹션
```

## 4. Hero 섹션

### 목적

- 첫 인상에서 브랜드의 급을 결정한다

### 구성

- 로고
- 메인 헤드라인
- 서브 카피
- CTA 2개
- 배경 비주얼 또는 그래픽 레이어

### 추천 헤드라인

- 분위기를 먼저 보고, 대화는 그 다음에

### 추천 서브 카피

- 욕망백서는 피드와 프로필을 통해 사람의 결을 먼저 읽고, 요청이 수락된 뒤에만 대화를 시작하는 익명 취향 플랫폼입니다.

### CTA

- 시작하기
- 분위기 둘러보기

## 5. 핵심 가치 섹션

### 목적

- 서비스가 왜 다른지 짧고 명확하게 설명

### 카드 3개 구성

#### 카드 1

- 제목: 피드로 분위기를 읽다
- 설명: 매칭 카드보다 먼저, 글과 분위기로 사람을 탐색합니다

#### 카드 2

- 제목: 요청은 명확하게
- 설명: 대화는 요청과 수락 이후에만 시작됩니다

#### 카드 3

- 제목: 상태와 환불은 숨기지 않다
- 설명: 요청, 거절, 만료, 환불까지 모든 흐름을 분명하게 보여줍니다

## 6. 어떻게 시작되는지 섹션

### 목적

- 사용 흐름을 한눈에 보여준다

### 4단계 구조

1. 프로필 만들기
2. 피드에서 분위기 읽기
3. 대화 요청 보내기
4. 수락 후 대화 시작

## 7. 포인트 / 환불 설명 섹션

### 목적

- 과금에 대한 불신을 선제적으로 줄인다

### 문구

- 새로운 대화 요청에만 포인트가 사용됩니다
- 상대가 거절하면 45P가 환불됩니다
- 24시간 내 응답이 없으면 전액 환불됩니다
- 수락 후 대화는 무료입니다

### 여성 메리트 박스

- 여성 회원은 가입 시 270P를 지급받습니다
- 받은 요청을 수락하고 대화하는 데에는 포인트가 들지 않습니다

## 8. 안전 / 운영 원칙 섹션

### 목적

- 서비스가 방치된 공간이 아니라는 인상을 준다

### 핵심 문구

- 원치 않는 요청은 언제든 거절하거나 차단할 수 있습니다
- 외부 연락처 요구, 금전 요구, 협박은 신고 대상입니다
- 이 공간은 취향을 존중하지만, 타인의 경계를 침범할 수는 없습니다

## 9. 최종 CTA 섹션

### 목적

- 가입 또는 탐색으로 마무리 전환

### 문구 후보

- 내 분위기를 소개하고 시작하기
- 먼저 둘러보고, 필요할 때 요청하세요

### 버튼

- 시작하기
- 홈 둘러보기

## 10. 비주얼 방향

- 다크 배경 위에 얇은 타이포와 깊이 있는 레이어
- 노골적 인물컷보다 추상적이고 무드 중심인 배경
- 편집지 같은 레이아웃 밀도
- 붉은색은 포인트로만 사용

## 11. 피해야 할 랜딩 표현

- 지금 바로 뜨거운 대화
- 야한 사람들과 즉시 연결
- 외로운 밤 해결
- 핫한 회원 다수 대기중

## 12. 최종 판단

랜딩 페이지는 욕망백서를 설명하는 페이지가 아니라  
욕망백서의 공기를 먼저 체감하게 하는 페이지여야 한다.

보고 나면 `야하다`보다 `정제돼 있다`가 먼저 남아야 한다.
