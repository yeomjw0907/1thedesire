# 욕망백서 화면별 상세 기능명세 v0.1

## 1. 랜딩

### 목적

- 서비스 소개
- 성인 서비스 유의사항 전달
- 로그인 진입

### 필수 구성

- 브랜드 로고
- 한 줄 가치 제안
- 시작하기 버튼
- 19세 이상 안내
- 약관 및 정책 링크

## 2. 로그인 및 가입

### 목적

- 빠른 진입
- 최소 필수 정보 수집

### 필수 기능

- 간편 로그인
- 닉네임 입력
- 성별 선택
- 연령대 선택
- 지역 선택
- 성향 입력
- 자기소개 입력
- `19세 이상입니다` 체크
- 약관 동의 체크
- 가입 완료 버튼

### 예외 처리

- 닉네임 중복
- 자기소개 미입력
- 필수 동의 누락

### 가입 완료 후 처리

- 프로필 생성
- 여성 회원 가입 시 270P 지급
- 기본 소개글 자동 작성 및 발행
- 홈 피드 진입

## 3. 홈 피드

### 목적

- 실시간 탐색
- 유저 체류 시간 확보

### 필수 기능

- 최신순 게시글 목록
- 작성 시간 노출
- 태그 및 성향 정보 표시
- 글쓰기 플로팅 버튼
- 게시글 클릭 시 프로필 이동
- 필터: 전체 / 신규

## 4. 글쓰기

### 목적

- 피드 활성화

### 필수 기능

- 텍스트 입력
- 이미지 1장 업로드
- 게시 버튼
- 삭제 버튼
- 수정 버튼

## 5. 검색

### 목적

- 관심 유저 빠른 탐색

### 필수 기능

- 닉네임 검색
- 지역 필터
- 성향 필터
- 태그 검색

### 결과 카드

- 닉네임
- 연령대
- 지역
- 성향
- 최근 활동 여부

## 6. 프로필 상세

### 목적

- 상대 정보 확인
- DM 진입

### 필수 기능

- 닉네임
- 성별
- 연령대
- 지역
- 성향
- 자기소개
- 작성 게시글 목록
- DM 요청 버튼
- 신고 버튼
- 차단 버튼

## 7. DM 요청 팝업

### 목적

- 과금과 합의 프로세스 시작

### 필수 기능

- `90P 차감` 안내
- 환불 규칙 요약
- 확인 버튼
- 취소 버튼

### 필수 문구

- 요청 즉시 90P가 차감됩니다
- 상대가 거절하면 50% 환불됩니다
- 상대가 24시간 내 응답하지 않으면 전액 환불됩니다
- 상대가 차단하면 환불되지 않습니다
- 상대가 수락해야 대화가 시작됩니다

## 8. 합의 대기 화면

### 목적

- 요청 상태 관리

### 상태

- 대기중
- 수락됨
- 거절됨
- 만료됨

### 필수 기능

- 상태 표시
- 요청 시각 표시
- 상대 응답 대기 안내
- 환불 결과 안내

## 9. 채팅방

### 목적

- 합의 이후 대화

### 필수 기능

- 메시지 목록
- 메시지 입력
- 전송
- 신고
- 차단
- 상대 프로필 바로가기

### 제약

- 수락 이전에는 입력창 비활성
- 차단 후 대화 불가
- 수락 이후에는 추가 포인트 없이 대화 가능

## 10. DM 목록

### 목적

- 진행 중 요청과 대화 관리

### 탭 구성 권장

- 대기

- 종료

### 카드 정보

- 상대 닉네임
- 상태
- 요청 시간
- 환불 여부

## 11. 포인트

- 거래 내역
