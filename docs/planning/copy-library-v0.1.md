# 욕망백서 브랜드 카피 / UI 문구 라이브러리 v0.1

## 1. 목적

- 욕망백서의 브랜드 톤을 실제 서비스 문구로 일관되게 적용한다.
- 화면, 버튼, 상태, 안내 문구가 싸구려 성인앱처럼 보이지 않도록 기준을 만든다.

## 2. 기본 문체 원칙

### 해야 하는 말투

- 차분하고 명확할 것
- 짧지만 맥락이 있을 것
- 과장하지 않을 것
- 존중과 통제감을 줄 것

### 하지 말아야 할 말투

- 노골적인 유혹형
- 자극적인 훅업형
- 싼 티 나는 광고형
- 법적 공포를 앞세우는 경고형

### 좋은 문장 방향

- 분위기를 먼저 읽고, 대화는 그 다음에
- 취향은 숨기지 말고, 안전하게 나누세요
- 요청은 신중하게, 대화는 명확하게

## 3. 브랜드 핵심 카피

### 메인 카피 후보

- 합의된 대화만 시작되는 익명 취향 플랫폼
- 분위기를 먼저 보고, 대화는 그 다음에 시작하세요
- 은밀하지만 불안하지 않은 공간
- 취향은 더 세련되게, 대화는 더 명확하게

### 서브 카피 후보

- 피드에서 분위기를 읽고, 프로필에서 판단한 뒤, 요청하세요
- 상대의 결을 먼저 보고 대화를 시작할 수 있습니다
- 명확한 상태와 환불 규칙으로 더 안심되는 대화 경험을 제공합니다

## 4. 랜딩 문구

### 헤드라인 후보

- 취향을 안전하게 나누는 가장 세련된 방법
- 익명성은 숨김이 아니라 통제감이어야 합니다
- 대화는 가볍게 열리지 않아야 더 편안합니다

### 설명 문구 후보

- 욕망백서는 피드와 프로필을 통해 분위기를 먼저 읽고, 합의된 요청으로만 대화를 시작하는 취향 기반 익명 플랫폼입니다.
- 이곳의 대화는 갑자기 열리지 않습니다. 먼저 보고, 판단하고, 요청한 뒤 시작됩니다.

### CTA

- 시작하기
- 분위기 둘러보기
- 프로필 만들기

## 5. 가입 / 온보딩 문구

### 기본 안내

- 기본 정보를 입력하고 내 분위기를 소개해보세요
- 자기소개는 짧아도 좋지만, 나를 판단할 수 있을 만큼은 필요합니다
- 가입이 완료되면 기본 소개글이 자동으로 등록됩니다

### 필드 문구

- 닉네임: `이 공간에서 사용할 이름`
- 지역: `너무 자세하지 않게 설정해도 괜찮습니다`
- 성향: `나를 설명하는 취향 한두 가지`
- 자기소개: `대화를 시작하기 전에 보여주고 싶은 분위기`

### 체크박스 문구

- 본인은 19세 이상입니다
- 운영정책 및 이용약관에 동의합니다

### 가입 완료 후

- 이제 분위기를 둘러볼 수 있습니다
- 첫 소개글이 등록되었습니다. 언제든 수정하거나 삭제할 수 있습니다

## 6. 홈 피드 문구

### 탭

- 전체
- 신규

### 빈 상태

- 아직 올라온 글이 없습니다
- 첫 분위기를 남겨보세요

### 글쓰기 유도

- 지금 떠오른 분위기를 남겨보세요
- 대화는 글 한 줄에서 시작되기도 합니다

### 카드 보조 문구

- 프로필 보기
- 이 사람 더 보기

## 7. 프로필 문구

### 프로필 섹션 타이틀

- 소개
- 최근 분위기
- 성향

### DM 안내 문구

- 요청 시 90P가 차감됩니다
- 상대가 거절하면 45P가 환불됩니다
- 24시간 내 응답이 없으면 전액 환불됩니다
- 수락 후 대화는 무료입니다

### 여성 유저 대상 보조 문구

- 받은 요청을 수락하고 대화하는 데에는 포인트가 들지 않습니다
- 원하지 않는 요청은 언제든 거절하거나 차단할 수 있습니다

## 8. DM 요청 문구

### 팝업 제목

- 대화 요청 보내기
- 이 사람에게 먼저 말을 걸어볼까요

### 정책 설명

- 요청을 보내면 90P가 차감됩니다
- 상대가 거절하면 45P가 환불됩니다
- 24시간 내 응답이 없으면 전액 환불됩니다
- 수락 후 대화는 무료로 이어집니다

### 버튼

- 취소
- 요청 보내기

## 9. DM 상태 문구

### 대기중

- 상대의 응답을 기다리는 중입니다
- 아직 대화는 시작되지 않았습니다

### 수락됨

- 대화를 시작할 수 있습니다
- 이제 자유롭게 대화를 이어가세요

### 거절됨

- 요청이 거절되었습니다
- 45P가 환불되었습니다

### 만료됨

- 응답 없이 요청이 종료되었습니다
- 90P가 전액 환불되었습니다

### 차단됨

- 이 대화는 더 이상 이어갈 수 없습니다
- 상대를 존중하며 다음 대화를 찾아보세요

## 10. 포인트 / 환불 문구

### 잔액

- 현재 보유 포인트

### 내역

- 충전 내역
- 사용 내역
- 환불 내역

### 안내 문구

- 새로운 대화 요청에만 포인트가 사용됩니다
- 수락된 대화는 추가 과금 없이 이어집니다
- 상태에 따라 환불이 자동 반영됩니다

## 11. 신고 / 차단 문구

### 신고

- 이 사용자를 신고할까요
- 불편하거나 부적절한 행동이 있었다면 알려주세요

### 차단

- 이 사용자를 차단할까요
- 차단하면 서로의 프로필과 대화가 제한됩니다

### 완료 문구

- 신고가 접수되었습니다
- 차단이 적용되었습니다

## 12. 운영 / 안전 문구

### 짧은 안전 메시지

- 원치 않는 요청은 언제든 거절하거나 차단할 수 있습니다
- 외부 연락처 요구, 금전 요구, 협박은 신고 대상입니다
- 이 공간은 취향을 존중하지만, 타인의 경계를 침범할 수는 없습니다

### 정책 요약 문구

- 요청은 신중하게, 대화는 명확하게
- 상태와 환불 규칙은 숨기지 않습니다
- 익명성은 사용자의 통제감에서 시작됩니다

## 13. 금지해야 할 문구 예시

아래 같은 톤은 사용하지 않는다.

- 지금 바로 핫한 대화 시작
- 외로운 밤 바로 해결
- 야한 사람들과 즉시 연결
- 자극적인 만남을 지금 바로

## 14. 카피 사용 체크리스트

- 이 문구가 유치하지 않은가
- 이 문구가 사용자에게 통제감을 주는가
- 이 문구가 너무 노골적이지 않은가
- 이 문구가 서비스 규칙을 숨기지 않는가
- 이 문구가 브랜드를 더 세련되게 만드는가
