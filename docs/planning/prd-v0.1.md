# 욕망백서 PRD v0.1

## 1. 제품 개요

### 1-1. 제품 한 줄 정의

욕망백서는 취향 기반 익명 소셜 피드와 상호 합의형 1:1 대화를 결합한 모바일 중심 플랫폼이다.

### 1-2. 비전

은밀한 욕망을 가장 안전하고 세련되게 해소하는 대한민국 No.1 취향 존중 플랫폼

### 1-3. 핵심 가치 제안

- 익명성과 취향 존중을 전제로 한 성인 커뮤니티 경험 제공
- 성적 대화 시작 전 상호 합의 절차를 통해 심리적 안전감 제공
- 기존 브랜드 자산을 활용해 빠르게 초기 유저 풀 확보
- 1인 운영이 가능한 수준까지 결제, 운영, 신고 흐름 자동화

## 2. 문제 정의

### 2-1. 현재 시장 문제

- 범용 데이팅 앱은 구체적 취향 기반 대화를 지원하지 못함
- 트위터류 플랫폼은 익명성은 높지만 운영 품질과 안전장치가 약함
- 성적 대화의 경계와 합의가 불명확해 유저 불안이 큼
- 기존 유사 서비스는 UI 완성도, 결제 자동화, 운영 시스템이 낙후됨

### 2-2. 우리가 해결하려는 핵심 문제

- 취향 기반 성인 유저가 안전하게 탐색하고 대화를 시작할 수 있어야 함
- 유저가 대화를 시작하기 전에 상대의 의사를 확인할 수 있어야 함
- 대표 1인이 운영 가능한 수준으로 백오피스 부담을 낮춰야 함

## 3. 목표와 비목표

### 3-1. MVP 목표

- 성인 유저가 가입 후 프로필을 만들고 피드를 볼 수 있다
- 유저가 글을 작성하고 다른 유저를 탐색할 수 있다
- 유저가 포인트를 사용해 대화 요청을 보낼 수 있다
- 양측 동의 후에만 채팅이 가능하다
- 포인트 충전과 차감 이력이 기록된다
- 신고와 차단이 최소 수준으로 동작한다

### 3-2. 비목표

- 영상 통화
- 정교한 추천 알고리즘
- 데스크톱 최적화
- 대규모 관리자 조직을 전제로 한 수동 운영
- 정식 앱스토어 앱 출시

## 4. 타겟 유저

### 4-1. 1차 타겟

- 20대 후반에서 30대 후반 성인
- 익명성을 중요하게 여김
- FWB, BDSM 등 특정 취향 기반 대화 니즈가 있음
- 일반 데이팅 앱보다 더 직접적이고 선명한 의사 표현을 원함

### 4-2. 유저 동기

- 같은 취향의 사람을 더 빨리 찾고 싶다
- 대화 전 상대의 의사를 먼저 확인하고 싶다
- 야한 사이트 느낌이 아니라 세련된 공간에서 활동하고 싶다

## 5. 핵심 원칙

- 모바일 우선
- 익명성 우선
- 합의 기반 대화
- 저질스러움보다 세련된 욕망
- 운영 자동화 우선
- 법적 안전을 과장하지 않고, 분쟁 시점의 기록 보조 장치를 제공

## 6. 주요 가설

- 인스타 기반 기존 팬덤은 초기 가입 전환에 의미 있는 도움이 된다
- 유저는 단순 매칭보다 피드 기반 탐색에 더 오래 머문다
- 대화 시작 전 합의 단계가 있어도 전환율이 크게 떨어지지 않는다
- 포인트 선차감 구조도 UI와 정책이 명확하면 수용 가능하다

## 7. 핵심 정책 결정

### 7-1. 가입 즉시 자동 포스팅

- 가입 완료 시 시스템이 첫 글을 자동 게시한다
- 기본 문구 예시: `안녕하세요. [지역] 거주 [성향]입니다.`
- 가입 중 관련 고지 문구를 노출한다
- 자동 생성 게시글은 이후 수정 또는 삭제 가능하다

### 7-2. DM 시작 전 90P 선차감

- DM 요청 시 90P를 즉시 차감한다
- 상대방이 합의 버튼을 누르기 전까지는 채팅 입력 불가
- 상대방이 수락하면 채팅방이 활성화된다
- 상대방이 거절하면 45P를 환불한다
- 상대방이 24시간 내 미응답이면 요청이 만료되고 90P를 전액 환불한다
- 상대방이 차단한 경우 환불은 없다

### 7-3. 성별별 과금 정책

- 새로운 DM 요청을 보내는 행위에는 공통적으로 포인트가 필요하다
- 여성 회원은 가입 시 270P를 기본 지급한다
- 여성 회원은 받은 DM을 수락하고 대화를 시작할 때 추가 포인트가 들지 않는다
- 남성 회원도 받은 DM을 수락하고 이미 열린 채팅방에서 답장할 때는 추가 포인트가 들지 않는다
- 수락 이후 채팅방 내 메시지 송수신은 무료다

### 7-4. 성인 인증

- 가입 시 `본인은 19세 이상입니다` 체크박스를 둔다
- 약관 및 금지행위 동의는 필수다
- 운영상 의심 계정은 별도 검토 대상으로 관리한다

### 7-5. 가입 정보

- 회원가입 시 자기소개 입력은 필수다
- 닉네임, 성별, 연령대, 지역, 성향, 자기소개를 입력받는다

## 8. 사용자 여정

### 8-1. 가입

1. 랜딩 진입
2. 서비스 소개 및 유의사항 확인
3. 간편 로그인
4. 닉네임, 성별, 연령대, 지역, 성향, 자기소개 입력
5. `19세 이상`, 약관, 금지행위 동의 체크
6. 가입 완료
7. 여성 회원은 270P를 기본 지급받음
8. 시스템이 기본 소개글 자동 게시
9. 유저는 홈 피드로 진입

### 8-2. 피드 탐색

1. 홈 피드에서 최신 글 확인
2. 지역, 성향, 태그로 탐색
3. 게시글 또는 프로필 클릭
4. 프로필에서 대화 요청 또는 차단, 신고 가능

### 8-3. DM 요청

1. 유저가 DM 버튼 클릭
2. `90P 차감` 안내 팝업 노출
3. 유저 확인 시 포인트 차감 및 요청 생성
4. 상대방에게 `합의 요청` 상태로 노출
5. 상대방이 수락하면 채팅방 활성화
6. 상대방이 거절하면 45P 환불
7. 상대방이 미응답이면 만료 처리 후 90P 전액 환불
8. 수락 이후 채팅은 무료

### 8-4. 채팅

1. 양측 합의 완료
2. 메시지 입력창 활성화
3. 신고와 차단은 채팅방에서 가능
