# 욕망백서 프로필 정보 구조 상세 문서 v0.1

## 1. 목적

- 프로필을 단순 사용자 정보 묶음이 아니라 `대화 전 판단 도구`로 설계한다.
- 어떤 정보를 어떤 순서로 보여줘야 DM 전환과 신뢰에 도움이 되는지 정리한다.

## 2. 프로필의 역할

욕망백서에서 프로필은 소개 페이지가 아니다.  
이 사람과 대화를 시작할지 말지를 결정하는 `판단 화면`이다.

따라서 프로필은 아래 세 가지를 동시에 만족해야 한다.

- 빠르게 읽힌다
- 충분한 맥락을 준다
- 과도하게 노출되지 않는다

## 3. 정보 우선순위

### 1순위: 판단에 바로 필요한 정보

- 닉네임
- 연령대
- 지역
- 성별
- 성향
- 자기소개

### 2순위: 분위기를 보강하는 정보

- 최근 활동
- 최근 게시글
- 성향 태그
- DM 가능 상태

### 3순위: 확장 정보

- 프로필 이미지
- 공개 범위 설정
- 선호/비선호 태그
- 활동 패턴

## 4. 프로필 레이아웃 구조

```text
[프로필 헤더]
닉네임
연령대 · 지역 · 성별
성향

[상태 라인]
최근 활동
DM 가능 상태
보조 안내

[자기소개]
짧지만 충분한 본문

[성향 태그]
구조화된 태그 목록

[최근 분위기]
최근 게시글 리스트

[하단 CTA]
대화 요청 보내기
```

## 5. 필드별 설계 기준

### 5-1. 닉네임

역할:

- 첫인상
- 익명성 유지

원칙:

- 과도한 장식 문자 제한 검토
- 읽기 어려운 닉네임은 지양

### 5-2. 연령대

역할:

- 실제 나이 공개 부담을 줄이면서 맥락 제공

권장 표현:

- 20대 후반
- 30대 초반
- 30대 후반

### 5-3. 지역

역할:

- 거리감과 생활권 감각 제공

원칙:

- 너무 세밀하지 않게
- `서울`, `경기`, `부산` 같은 광역 단위 우선

### 5-4. 성별

역할:

- 탐색 기준의 하나

원칙:

- 기본 정보로 두되 과도하게 부각하지 않음

### 5-5. 성향

역할:

- 가장 중요한 판단 재료

원칙:

- 자유입력만 두지 말고 구조화 가능한 설계를 고려
- 짧고 읽기 쉬워야 함

예:

- Sub
- Dom
- Switch
- FWB

### 5-6. 자기소개

역할:

- 이 사람의 말투, 분위기, 거리감을 판단하게 함

원칙:

- 너무 길 필요는 없지만 공란은 안 됨
- 외모보다 태도와 분위기가 읽히는 문장이 좋음

좋은 예:

- 서로 말투와 분위기가 맞는지 먼저 보는 편입니다
- 빠른 대화보다 결이 맞는 사람이 좋습니다

나쁜 예:

- 외로워요 연락주세요
- 바로 가능

## 6. 성향 태그 구조 제안

### 목적

- 자유입력만으로는 탐색 품질이 떨어지기 때문에 구조화 태그를 일부 도입한다

### 초기 태그 범주 예시

- 역할: Dom / Sub / Switch
- 관계 성향: FWB / 탐색중 / 대화우선
- 분위기: 조용함 / 솔직함 / 천천히 / 직설적

원칙:

- 너무 많은 태그를 초기에 강요하지 않음
- 2~4개 선택 정도가 적절

## 7. 최근 게시글의 역할

프로필에서 최근 글은 보조 기능이 아니다.  
실제로는 자기소개보다 더 많은 정보를 줄 수 있다.

역할:

- 말투 확인
- 현재 분위기 확인
- 피드와 프로필 연결

권장 노출:

- 최근 3~5개
- 제목 없이 본문 일부 미리보기

## 8. DM 가능 상태 표시

### 목적

- 요청 전 불확실성 감소

표시 예시:

- 요청 가능
- 현재 대화에 신중한 편
- 받은 요청 수락 무료

원칙:

- 강한 필터 문구보다 부드러운 안내형으로

## 9. 프로필 이미지 정책 초안

### 원칙

- 이미지가 없어도 프로필이 충분히 성립해야 한다
- 사진은 판단 보조 요소다

### 권장 방향

- 초기 MVP에서는 선택 요소
- 얼굴 노출이 없어도 분위기 컷 가능
- 과도한 노출 사진은 운영정책 대상

## 10. 프로필에서 숨겨야 할 것

- 과한 배지
- 랭킹
- 과금 중심 정보
- 과도한 성적 노출 문구

## 11. 프로필 성공 기준

- 사용자가 10초 안에 이 사람의 결을 읽을 수 있다
- DM을 보내도 될지 판단할 수 있다
- 차단/신고가 필요할 때 찾기 쉽다
- 사진이 없어도 정보가 빈약하지 않다

## 12. 최종 판단

욕망백서 프로필은 `예쁜 카드`가 아니라  
`대화 전 신중한 판단을 돕는 취향 보고서`여야 한다.

이 기준이 흔들리면 서비스가 금방 평범한 성인앱처럼 보이게 된다.
