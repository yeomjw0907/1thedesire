# 욕망백서 운영 KPI / 대시보드 설계 v0.1

## 1. 목적

- 욕망백서의 출시 후 운영 상태를 한눈에 볼 수 있는 KPI 구조를 정의한다.
- 대표 1인이 매일, 매주, 매월 봐야 할 지표를 구분한다.

## 2. KPI 설계 원칙

- 숫자는 많기보다 의미 있어야 한다
- 제품 성장보다 `신뢰와 안정성` 지표를 먼저 본다
- 리텐션, 과금, 신고, 환불은 함께 봐야 한다
- KPI는 운영 액션으로 이어질 수 있어야 한다

## 3. 대시보드 구조

```text
Top Summary
-> 유입 / 가입
-> 탐색 / 활동
-> DM / 전환
-> 포인트 / 환불
-> 신고 / 차단 / 제재
-> 리텐션
```

## 4. Top Summary

매일 가장 먼저 보는 숫자:

- 신규 가입 수
- Day 1 재방문율
- DM 요청 수
- DM 수락률
- 환불 발생 수
- 신고 접수 수
- 차단 발생 수

## 5. 유입 / 가입 지표

### 핵심 지표

- 방문 수
- 가입 시작 수
- 가입 완료 수
- 가입 완료율
- 자동 소개글 생성 수
- 자동 소개글 삭제율

### 중요 이유

- 가입 퍼널이 막히면 전체가 멈춘다
- 자동 소개글 삭제율은 첫 인상 거부감을 보여주는 신호다

## 6. 탐색 / 활동 지표

### 핵심 지표

- 홈 피드 열람 수
- 피드 평균 체류 시간
- 프로필 상세 진입률
- 게시글 작성률
- 프로필 보완율

### 중요 이유

- 욕망백서는 매칭앱보다 탐색앱 성격이 강하다
- 피드와 프로필이 살아 있어야 DM도 살아난다

## 7. DM / 전환 지표

### 핵심 지표

- DM 요청 수
- 요청 대비 수락률
- 요청 대비 거절률
- 요청 대비 만료율
- 수락 후 첫 메시지 전송률
- 첫 DM 요청까지 걸린 평균 시간

### 중요 이유

- DM은 가장 중요한 유료 행동이다
- 수락률과 만료율은 프로필 품질, 피드 품질, 과금 UX 문제를 동시에 드러낸다

## 8. 포인트 / 환불 지표

### 핵심 지표

- 총 충전 건수
- 총 충전 금액
- 유료 전환율
- 평균 결제 금액
- 환불 발생 건수
- 환불 비율
- 여성 가입 보너스 사용률

### 중요 이유

- 과금이 잘 되느냐보다 `과금 구조를 받아들이고 있느냐`가 중요하다
- 환불 비율이 과도하면 UX 또는 정책 문제가 있다는 신호다

## 9. 신고 / 차단 / 제재 지표

### 핵심 지표

- 신고 접수 수
- 신고 유형별 비율
- 차단 발생 수
- 계정 제한 수
- 일시정지 수
- 영구정지 수

### 중요 이유

- 안전감은 실제 신고/차단 패턴으로 보인다
- 차단이 많고 신고가 적으면 UI 접근성 또는 신고 유도 문구를 점검해야 한다

## 10. 리텐션 지표

### 핵심 지표

- Day 1 / Day 3 / Day 7 재방문율
- 첫 7일 내 프로필 보완율
- 첫 7일 내 첫 DM 요청 경험률
- 첫 7일 내 첫 DM 수락 경험률

### 중요 이유

- 욕망백서는 첫 7일 적응이 핵심이다
- 이 구간에서 정착하지 못하면 과금도 일어나지 않는다

## 11. 여성 / 남성 분리 관점

### 여성 유저

- 가입 완료율
- Day 1 재방문율
- 받은 DM 수락률
- 차단률
- 신고율
- 270P 사용률

### 남성 유저

- 가입 완료율
- 첫 충전 전환율
- DM 요청 수
- 수락률
- 거절률
- 만료율

### 왜 분리해서 봐야 하는가

- 현재 서비스 구조상 여성과 남성의 첫 경험이 다르다
- 한쪽 문제를 다른 쪽 전체 평균이 가릴 수 있다

## 12. 주기별 확인 항목

### 매일

- 신규 가입
- DM 요청 / 수락 / 거절 / 만료
- 환불 건수
- 신고 / 차단

### 매주

- Day 1 / Day 7 리텐션
- 프로필 보완율
- 자동 소개글 삭제율
- 첫 충전 전환율

### 매월

- 총 유입
- 총 결제
- 여성/남성별 행동 차이
- 제재 패턴 변화

## 13. 위험 신호

- 자동 소개글 삭제율 급증
- DM 만료율 급증
- 환불 비율 급증
- 여성 차단률 급증
- 신고 유형 중 금전 요구/협박 비율 증가

## 14. 권장 대시보드 카드 문구

- 오늘 신규 가입
- 오늘 대화 요청
- 오늘 환불 발생
- 미처리 신고
- Day 1 재방문율
- 자동 소개글 삭제율

## 15. 최종 판단

욕망백서의 KPI는 단순 성장 지표가 아니라  
`이 서비스가 믿을 만한 흐름으로 굴러가고 있는가`를 보는 지표여야 한다.

특히 DM, 환불, 차단, 리텐션은 한 묶음으로 봐야 한다.
