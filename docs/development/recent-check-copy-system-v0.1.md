# 최근검사 확인 Copy System v0.1

## 목적
- STI 신뢰 기능의 사용자 노출 명칭과 카피를 `최근검사 확인`으로 통일한다.
- 이 문서는 실제 UI, 정책 문구, 기능 설명, 관리자 처리 결과 메시지에 들어갈 텍스트 기준안이다.

## 최종 네이밍 결정

### 사용자 노출 메인명
- `최근검사 확인`

### 설명용 확장 표현
- `최근 STI 검사 확인 정보를 공개한 프로필`

### 법무/고지용 정확 표현
- `최근 STI 검사 확인`

## 네이밍 원칙
- 바깥에 보이는 이름은 짧고 읽기 쉬워야 한다.
- 설명 문구에서는 `STI`를 명시해 의미를 분명히 한다.
- `안전`, `무감염`, `보장` 같은 표현은 사용하지 않는다.

## 허용 문구
- `최근검사 확인`
- `최근 STI 검사 확인 정보를 공개한 프로필입니다`
- `검사 시점 기준 정보입니다`
- `현재 상태를 보증하지 않습니다`
- `만남 전 추가 확인과 보호수단 사용을 권장합니다`

## 금지 문구
- `안전 인증`
- `성병 없음 인증`
- `무감염 인증`
- `안심 상대`
- `믿고 만나도 됨`
- `문제 없는 사람`
- `100% 안전`

## 1. 프로필 화면 카피

### 배지 칩
- `최근검사 확인`

### 배지 보조 문구
- `검사일 2026.03.02`
- `2026.04.01까지 표시`

### 신뢰 정보 섹션 설명
- `최근 STI 검사 확인 정보를 공개한 프로필입니다`

### 배지 상세 모달 제목
- `최근검사 확인`

### 배지 상세 모달 본문
- `이 프로필은 최근 STI 검사 확인 정보를 공개하고 있어요.`
- `검사 시점 기준의 정보이며 현재 상태를 보증하지 않습니다.`
- `만남 전에는 대화와 보호수단 여부도 함께 확인해 주세요.`

## 2. DM 요청 화면 카피

### 배지 노출 문구
- `최근검사 확인 정보 공개 중`

### 보조 설명
- `검사일 2026.03.02`
- `검사 시점 기준 정보예요`

### 상세 안내 문구
- `최근검사 확인 정보는 신뢰를 돕는 참고 정보예요.`
- `현재 상태를 보증하지 않으니 만남 전 대화와 보호수단 여부를 함께 확인해 주세요.`

## 3. 제출 진입 화면 카피

### 제목
- `최근검사 확인`

### 설명
- `최근 STI 검사 확인 정보를 프로필에 표시할 수 있어요.`
- `검증이 완료되면 공개 여부를 직접 선택할 수 있어요.`
- `이 정보는 검사 시점 기준이며 현재 상태를 보증하지 않습니다.`

### 체크박스 설명
- `민감정보 수집·이용에 동의합니다`
- `검수 목적 처리에 동의합니다`
- `검증 완료 시 프로필에 공개하는 것에 동의합니다`

## 4. 제출/상태 화면 카피

### 제출 버튼
- `검사 정보 제출`

### 제출 완료
- 제목: `제출이 완료됐어요`
- 본문: `보통 24~48시간 안에 검수가 진행돼요.`

### 검수 대기
- 제목: `검수 대기 중`
- 본문: `제출한 정보를 순서대로 확인하고 있어요.`

### 검토 중
- 제목: `검토 중`
- 본문: `운영자가 제출한 정보를 확인하고 있어요.`

### 승인 완료
- 제목: `최근검사 확인 완료`
- 본문: `이제 공개 여부를 직접 선택할 수 있어요.`

### 반려
- 제목: `이번 제출은 반려됐어요`
- 본문: `사유를 확인하고 다시 제출할 수 있어요.`

### 만료
- 제목: `표시 기간이 만료됐어요`
- 본문: `원하면 다시 검증을 진행할 수 있어요.`

### 철회
- 제목: `공개를 중단했어요`
- 본문: `프로필에서 최근검사 확인 배지가 숨겨졌어요.`

## 5. 공개 토글 카피

### 라벨
- `프로필에 공개`

### 보조 설명
- `검증 완료 후에도 공개 여부는 직접 선택할 수 있어요.`

### 공개 ON 완료
- `최근검사 확인 배지가 프로필에 표시돼요`

### 공개 OFF 완료
- `프로필에서 배지가 숨겨져요`

## 6. 반려 사유 카피 톤

### 원칙
- 공격적이지 않게
- 문서 진위 판단을 단정하지 말고 제출 기준 미충족 중심으로 설명

### 예시
- `검사일을 확인하기 어려워 반려되었어요`
- `제출 형식을 다시 확인해 주세요`
- `확인 가능한 정보가 부족해 다시 제출이 필요해요`

## 7. 관리자 화면 카피

### 섹션명
- `최근검사 확인 검수`

### 리스트 상태
- `검수 대기`
- `검토 중`
- `승인 완료`
- `반려`
- `만료`

### 버튼
- `상세 보기`
- `승인`
- `반려`
- `만료 처리`

## 8. 최종 권장안
- 리스트/프로필 칩: `최근검사 확인`
- 설명 문구: `최근 STI 검사 확인 정보를 공개한 프로필입니다`
- 상세 고지: `검사 시점 기준 정보이며 현재 상태를 보증하지 않습니다`
