# 욕망백서 권한 및 관리자 인증 문서 v0.1

## 1. 목적

- 일반 사용자와 운영자 권한을 어떻게 분리할지 정리한다.

## 2. 기본 역할

- `user`
- `admin`
- `system`

## 3. 권한 원칙

- 일반 사용자는 본인 데이터와 참여 데이터만 접근
- 운영자는 신고, 제재, 포인트 예외 처리를 수행 가능
- 시스템은 자동 보너스, 만료, 환불, 알림 작업 수행

## 4. 권장 구현 방향

- Supabase auth + 별도 admin claim 또는 profiles role 필드 활용
- 운영자 콘솔 접근은 추가 검증 필요
- 운영 액션은 반드시 로그 기록

## 5. 운영자 가능 작업

- 신고 조회 및 상태 변경
- 게시글 숨김
- 계정 restrict / suspend / ban
- 포인트 수동 조정
- payment event 확인

## 6. 최종 판단

운영자 권한은 강하지만 좁게 줘야 한다.
